# 🌿 Accessibility Audit Demo – GreenScape Lawn & Landscape

This project is a **realistic small business website demo** used to showcase how accessibility issues are identified and fixed in accordance with **WCAG (Web Content Accessibility Guidelines)** and **ARIA best practices**.

It simulates the type of site a client might bring for an accessibility audit, including common real-world problems and live remediation examples.

---

## 🔍 What This Demo Shows

* Identification of common accessibility issues:

  * Missing `alt` text on images
  * Unlabeled form inputs
  * Poor semantic structure

* Visual highlighting of issues on the page

* Basic automated fixes applied via JavaScript

* A realistic business website layout for context

---

## ⚙️ Features

* ✅ Interactive **“Run Audit”** button
* ✅ **Live issue highlighting** (visual outlines)
* ✅ **Apply Fixes** simulation
* ✅ Responsive layout (desktop/mobile)
* ✅ Realistic UI/UX matching a typical small business site

---

## 🧪 Purpose

This is a **demonstration project**, not a production-ready website.

It is designed to:

* Show how accessibility audits work
* Educate clients on common compliance issues
* Serve as a portfolio piece for accessibility services

---

## ⚠️ Important Notes

* Some issues are intentionally included for demonstration purposes
* Fixes shown are simplified examples (real audits are more comprehensive)
